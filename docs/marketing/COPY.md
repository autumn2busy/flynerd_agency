# Fly Nerd Tech Marketing Copy Blocks

## Core Message
- **Anchor:** "Authenticity is the new Authority."
- **Support line:** Enterprise-grade AI automations that feel human—and convert.

## CTA Hierarchy
1. **Primary:** Book a Strategy Call (segment-specific variant)
2. **Secondary:** Get a Personalization Blueprint (segment-specific variant)

## Segment Variants
- **Franchise:** speed-to-lead, multi-location attribution, governance.
- **Member Hub:** onboarding depth, retention loops, personalized lifecycle.
- **Enterprise Ops:** compliance guardrails, observability, exec visibility.

## Proof Blocks
- KPI tiles must include:
  - metric label
  - claimed movement value
  - methodology disclosure (collapsible detail)
- Case snippet format:
  - Problem
  - Intervention
  - Result

## Style Guidance
- Keep claims short and operationally specific.
- Avoid abstract AI language without mechanism.
- Mention enterprise controls (policy, consent, audit) near conversion claims.
